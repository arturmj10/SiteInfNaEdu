Bios

É importante ter a BIOS atualizada para garantir a compatibilidade e o desempenho do sistema. Após baixar a BIOS, siga as instruções do fabricante para atualizá-la.

Passo 2: Coloque a imagem do sistema operacional no pendrive

  1. Insira o pendrive no computador.
  2. Baixe e instale um programa para criar uma mídia de instalação bootável, como o Balena Etcher.
  3. Abra o programa e selecione a imagem ISO que você baixou.
  4. Selecione o pendrive como o dispositivo para criar a mídia bootável e inicie o processo.

Passo 3: Configure o BIOS

Para acessar a BIOS, siga as instruções de acordo com o tipo de computador:

  • PC Desktop (Windows): Reinicie o computador e pressione a tecla Delete (DEL) repetidamente ou a tecla F2 assim que o computador inicializar. Algumas placas-mãe podem usar outras teclas, como F1 ou F10.
  • Laptop (Windows): O procedimento pode variar dependendo do fabricante do laptop, mas geralmente, você precisa reiniciar o laptop e pressionar uma das seguintes teclas: F2, F12, Delete (DEL), ESC ou Fn + F2.
  • Computadores Mac: Reinicie o computador e mantenha pressionada a tecla Command (⌘) e a tecla R ao mesmo tempo até que o logotipo da Apple apareça. Em seguida, selecione "Utilitários" > "Utilitário de Disco" > "EFI Boot".

Passo 4: Configure o boot pelo pendrive

Depois de entrar na BIOS, siga estas etapas para configurar o boot pelo pendrive:

  1. Procure por uma opção como "Boot Order", "Boot Sequence" ou "Boot Priority".
  2. Selecione o pendrive na lista de dispositivos de boot e mova-o para o topo da lista. Isso garante que o computador inicie a partir do pendrive.
  3. Salve as configurações e saia da BIOS. Geralmente, você pode fazer isso pressionando a tecla apropriada listada na BIOS para salvar e sair, como F10 ou ESC. Confirme que deseja salvar as alterações e sair.
  4. O computador vai reiniciar automaticamente. Certifique-se de que o pendrive esteja conectado.

Passo 5: Instalação do Sistema Operacional

Após o pendrive ser bootado, siga as instruções abaixo para instalar o Windows:

  1. Selecione o idioma, a hora e o layout do teclado e clique em "Avançar".
  2. Clique em "Instalar agora".
  3. Insira a chave do produto do Windows, se solicitado, e clique em "Avançar".
  4. Leia e aceite os termos de licença e clique em "Avançar".
  5. Selecione "Personalizada: Instalar apenas o Windows (avançado)".
  6. Selecione a unidade onde você deseja instalar o Windows e clique em "Avançar".
  7. Se o disco tiver partições existentes, você pode excluí-las selecionando cada uma e clicando em "Excluir". Isso limpará o disco completamente.
  8. Selecione a unidade não alocada e clique em "Avançar".
  9. O Windows será instalado. Siga as instruções na tela.
  10. Após a instalação, o computador será reiniciado e o Windows estará pronto para uso.

Linux

Se você optou por instalar o Linux, siga estes passos:

  1. Selecione a opção de inicialização pelo pendrive.
  2. Siga as instruções do instalador do Linux para selecionar o idioma, fuso horário e outras configurações.
  3. Quando chegar à parte de particionamento, você pode escolher entre várias opções. Se deseja limpar completamente o disco, selecione a opção para "apagar o disco" ou "usar todo o disco". Certifique-se de que esteja selecionando o disco correto.
  4. Prossiga com a instalação e siga as instruções na tela.
  5. Após a instalação, reinicie o computador e o Linux estará pronto para uso.

Formatar PC no Windows 10

Você pode formatar o PC diretamente nas configurações do Windows 10. Aqui está como fazer:

  1. Abra o menu Iniciar e clique em "Configurações" (ícone de engrenagem).
  2. Selecione "Atualização e Segurança".
  3. No menu lateral esquerdo, selecione "Recuperação".
  4. Clique em "Começar" em "Restaurar o PC" e siga as instruções na tela.

Formatar PC no Windows 11

No Windows 11, você também pode formatar o PC diretamente nas configurações. Veja como:

  1. Abra o menu Iniciar e clique em "Configurações" (ícone de engrenagem).
  2. Selecione "Sistema".
  3. No menu lateral esquerdo, selecione "Recuperação e reinstalação".
  4. Clique em "Restaurar o PC" e siga as instruções na tela.
